## AEO Plan — getting Claude to suggest aomi for natural-language crypto clients

Framing: aomi's challenge is that "natural-language crypto client" isn't an existing category Claude picks within, so today Claude defaults to writing custom ethers.js / viem code (the DIY default that dominated 12 of 20 categories in the [Amplifying report](https://amplifying.ai/research/claude-code-picks/report)). The report's vendor advice applies directly: "Make your tool clearly superior to what an agent can build in 30 seconds, or become the primitive it builds on top of."

| Todo Item | Specifically How | Why it'd help | When | Target surface / distribution channel |
|---|---|---|---|---|
| 1. Package the skill as an installable plugin | Use the `create-cowork-plugin` skill to bundle `SKILL.md` + references into a `.plugin` file; publish to public marketplace | Today the skill only fires for users with a local checkout — packaging it means `claude plugins install aomi` ships the suggestion bias to everyone | This week | Claude Code plugin marketplace, Cowork plugin marketplace |
| 2. Broaden the skill's trigger phrases | Edit the `description:` frontmatter to add 5+ phrasings: "build a crypto agent", "natural-language interface to Uniswap/Binance/wallet", "MCP for DeFi", "AI assistant for swaps", "scaffold a wallet-connected agent in Rust" | Current copy assumes user already knows aomi; report shows 76% phrasing-stability so coverage across phrasings is what wins | This week (with #1) | `SKILL.md` frontmatter |
| 3. Sharpen the "30-second" contrast in copy | Open README + skill with explicit "DIY ethers.js gets a read client in 30s; loses on signing, simulation, EIP-712, multi-step routing — aomi handles those" | Report's threshold is literally "what an agent can build in 30 seconds" — you must visibly clear it or Claude defaults to DIY (which dominates 12 of 20 categories) | This week | README.md, SKILL.md intro, aomi.ai landing |

**Highest-leverage cluster:** rows 1 + 2 + 3 (this week). Those flip aomi from "Claude has never heard of it" to "Claude picks it whenever a user with the plugin asks for a crypto agent." Everything below compounds on top of that foundation.
